What is the Display Property?

The CSS display property specifies the display behavior of an element. It defines how an element is rendered in the document flow.

Every HTML element has a default display value depending on its type.

display: block

selector { display: block; }

Block-level elements start on a new line and take up the full width available. They can have margins, padding, width, and height set.

Examples: <div>, <p>, <h1>-<h6>, <section>, <article>

display: inline

selector { display: inline; }

Inline elements do not start on a new line and only take up as much width as necessary. They cannot have width or height set, and top/bottom margins/padding don't affect other elements.

Examples: <span>, <a>, <strong>, <em>

display: inline-block

selector { display: inline-block; }

Inline-block elements are similar to inline elements but can have width, height, margins, and padding set. They flow with text and other inline elements but behave like blocks in terms of styling.

display: none

selector { display: none; }

The element is completely removed from the document flow. It doesn't take up any space and is not rendered.

This is different from visibility: hidden which hides the element but reserves its space in the layout.

display: table

.table { display: table; }
.table-row { display: table-row; }
.table-cell { display: table-cell; }

These values let you make elements behave like HTML table elements without actually using table markup. This can be useful for layouts that need table-like behavior.
